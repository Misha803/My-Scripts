Frequently Asked Questions

Why is Windows Defender blocking the app?
Because these tools are self-signed and modify system, Windows Defender sometimes flags them as unrecognized. If you are having trouble launching a tool, Click here for the quick fix!
Which tool should I download first?
We highly recommend starting with the WOA Toolbox. It serves as an all-in-one hub that allows you to easily access, run, and update all the other tools seamlessly directly from your device!
✨ QuickRotate V4 Cleanup & V6 Setup

As we have discontinued QuickRotate V4 and you're moving to the new QuickRotate V6, follow these steps to remove the old V4 from auto-start:

  1. Disable V4 Autostart: Open QuickRotate V4 → Go to Settings → Uncheck all checkboxes. This stops it from starting with Windows and removes its desktop shortcuts.
  2. Close the app.
  3. Delete the file: You can safely delete the old .exe and the folder here: %AppData%\QuickRotate
  4. Switch to V6: Start the new version!

🎧 Pro-tip: Don't forget to enable the "Minimize to Tray on Close" and shortcuts toggles in V6 so you don't lose your rotation toggle by accident! ❤️
